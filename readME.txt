## Проект "Учет расходов"

### ОСНОВНАЯ ИДЕЯ:

Основной целью данного SPA-приложения является предоставление пользователям возможности учета своих расходов. Пользователи могут легко вводить данные о расходах, указывая сумму и категорию. Приложение группирует расходы по категориям, обеспечивая удобный просмотр и фильтрацию по месяцам.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

1**SPA-приложение: приложение состоит из четырех страниц, реализованных с использованием React-компонентов, обеспечивая навигацию между различными частями приложения.

2**React-компоненты: основная структура приложения построена на React-компонентах, обеспечивая управление состоянием и передачу данных между частями приложения.

3**Формы и списки: используются контролируемые инпуты для ввода расходов и отображения списка записей. Реализована сортировка, фильтрация расходов по категориям, по месяцам, по дате совершения покупки.

4**Взаимодействие с собственным API-сервисом, реализованным через https://642ee23f2b883abc64198889.mockapi.io/purchases, для сохранения данных о расходах.

### ОСОБЕННОСТИ ПРОЕКТА:

- пользователи могут легко добавлять свои расходы, указывая дату совершения покупки, сумму и категорию, а затем просматривать их в удобной форме.

- расходы группируются по категориям, по месяцам, также есть сортировка по стоимости, что позволяет пользователям быстро оценить, в каких областях они тратят больше всего.

- возможность фильтрации расходов по месяцам обеспечивает удобство анализа затрат в разные периоды времени.
